# DAOD

域自适应目标检测早期代码的运行环境为pytorch 0.4或者0.4.1，可以运行这些代码的显卡最多支持到2080Ti。如今30，40系列显卡在市场上占主流位置。
由此，我们重写了早期的域自适应目标检测的相关代码，使之能在30系列显卡上能够运行。
